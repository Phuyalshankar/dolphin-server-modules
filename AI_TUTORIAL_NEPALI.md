## १. Dolphin AI के हो?

Dolphin AI एउटा **Cursor-level Autonomous Agent** हो जसले:

- ✅ **तपाईंको पूरो project पढ्छ** (semantic indexing)
- ✅ **Files लेख्छ, edit गर्छ** (permission लिएर)
- ✅ **Shell commands run गर्छ** (dangerous commands block गर्छ)
- ✅ **Conversation याद राख्छ** (`.dolphin/history.json`)
- ✅ **Multiple AI providers** support गर्छ (Gemini, Groq, OpenAI, Ollama)
- ✅ **Roman Nepali** मा कुरा गर्छ (Nepali language in English script)

**किन Dolphin AI?**
- Terminal बाटै code generate गर्न सकिन्छ
- API key हाले मात्र — install गर्न पर्दैन
- Free tier available (Gemini, Groq)
- Offline पनि चल्छ (Ollama)

## २. Setup — API Key

**कम्तीमा एउटा API key चाहिन्छ।** `.env` file मा राख्नुहोस्:

```env
# ── Cloud AI (एउटा मात्र चाहिन्छ) ─────────────────────────────

# Option 1: Google Gemini (Recommended — free tier available)
# Get key: https://aistudio.google.com/apikey
GEMINI_API_KEY=AIzaSy...your_gemini_key_here

# Option 2: Groq — Llama 3.3 (Ultra fast, free tier)
# Get key: https://console.groq.com/keys
GROQ_API_KEY=gsk_...your_groq_key_here

# Option 3: OpenAI (GPT-4o)
# Get key: https://platform.openai.com/api-keys
OPENAI_API_KEY=sk-...your_openai_key_here

# Universal alias (माथिका जुनसुकैलाई काम गर्छ)
DOLPHIN_AI_KEY=your_any_key_here

# ── Advanced (optional) ──────────────────────────────────────────
# Custom OpenAI-compatible endpoint (LMStudio, Ollama, etc.)
DOLPHIN_AI_BASE_URL=http://localhost:1234/v1
DOLPHIN_AI_MODEL=gemini-2.0-flash   # Model override

# ── Local AI (Free, Offline) ─────────────────────────────────────
# पहिले Ollama install गर्नुहोस्: https://ollama.com
USE_OLLAMA=true
OLLAMA_MODEL=gemma3:latest     # वा llama3, codellama, mistral
OLLAMA_BASE_URL=http://localhost:11434/v1
```

**Provider Priority (Dolphin automatically choose गर्छ):**
```
DOLPHIN_AI_KEY → GEMINI_API_KEY → GROQ_API_KEY → OPENAI_API_KEY → Ollama (local)
```

### Free Keys कसरी पाउने?

**Gemini (सबैभन्दा सजिलो):**
1. https://aistudio.google.com/apikey मा जानुहोस्
2. Google account ले sign in गर्नुहोस्
3. "Create API Key" थिच्नुहोस्
4. `.env` मा `GEMINI_API_KEY=your_key` राख्नुहोस्

**Groq (सबैभन्दा छिटो):**
1. https://console.groq.com मा जानुहोस्
2. Account बनाउनुहोस् (free)
3. "API Keys" → "Create API Key"
4. `.env` मा `GROQ_API_KEY=your_key` राख्नुहोस्

npx dolphin generate "create order management system with status tracking"

**Generated file (`ai-generated.js`) हेर्ने:**
```bash
cat ai-generated.js
```

**Example output:**
```js
// ai-generated.js — Generated by Dolphin AI
import { createDolphinServer } from 'dolphin-server-modules/server';
import { zodMiddleware } from 'dolphin-server-modules/middleware/zod';
import { z } from 'zod';

const paymentSchema = z.object({
  amount: z.number().positive('Amount must be positive'),
  userId: z.string().min(1, 'UserId required'),
  description: z.string().optional(),
});

export function setupPaymentRoutes(app, auth) {
  app.post('/api/payments',
    auth.requireAuth,
    zodMiddleware(paymentSchema),
    async (ctx) => {
      const { amount, userId, description } = ctx.body;
      // Payment processing logic here
      return {
        success: true,
        payment: { id: Date.now().toString(), amount, userId, status: 'pending' }
      };
    }
  );
}
```

## ६. Agent Tools Reference

AI agent ले यी **tools** प्रयोग गर्छ:

| Tool | काम |
|------|-----|
| `read` | File पढ्ने |
| `write` | File लेख्ने / बनाउने |
| `patch` | File को specific part replace गर्ने (precision edit) |
| `delete` | File / folder delete गर्ने |
| `list` | Directory को files list गर्ने |
| `shell` | Terminal command run गर्ने |
| `search_symbol` | Function / class project भरि खोज्ने |
| `done` | Task complete — final message |

**Safety Rules:**
- ⚠️ **write, delete, shell** गर्नु अगाडि तपाईंबाट permission माग्छ (y/n)
- 🔒 `.env`, `secret`, `password`, `key` files को लागि warning देखाउँछ
- 🚫 `rm -rf /`, `mkfs`, `dd if=` — dangerous commands block गर्छ
- 📝 सबै actions `.dolphin/history.json` मा log हुन्छ

## ९. Code मा AI Service Use गर्ने

**Server code भित्रै AI use गर्ने:**

```js
import { AIService } from 'dolphin-server-modules/services/ai-service.js';

const ai = new AIService({
  apiKey: process.env.DOLPHIN_AI_KEY || process.env.GEMINI_API_KEY,
  model: process.env.DOLPHIN_AI_MODEL,  // optional override
});

// Single request
app.post('/api/ai/generate', auth.requireAuth, async (ctx) => {
  const { prompt } = ctx.body;

  const response = await ai.request(
    prompt,
    'You are a helpful assistant for a Nepali e-commerce app.'
  );

  return { success: true, response };
});

// Streaming response
app.post('/api/ai/stream', auth.requireAuth, async (ctx) => {
  const { prompt } = ctx.body;

  ctx.res.setHeader('Content-Type', 'text/event-stream');
  ctx.res.setHeader('Cache-Control', 'no-cache');
  ctx.res.setHeader('Connection', 'keep-alive');

  await ai.requestStream(
    prompt,
    'You are a helpful assistant.',
    (chunk) => {
      ctx.res.write(`data: ${JSON.stringify({ chunk })}\n\n`);
    }
  );

  ctx.res.write('data: [DONE]\n\n');
  ctx.res.end();
});

// Conversation with history
const conversations = new Map(); // userId → history

app.post('/api/ai/chat', auth.requireAuth, async (ctx) => {
  const userId = ctx.req.user.id;
  const { message } = ctx.body;

  // User को conversation history load
  if (!conversations.has(userId)) {
    conversations.set(userId, new AIService({
      apiKey: process.env.DOLPHIN_AI_KEY,
    }));
  }

  const userAI = conversations.get(userId);
  const response = await userAI.request(
    message,
    'You are a helpful customer service AI for our e-commerce shop.'
  );

  return { success: true, response };
});

// History clear
app.post('/api/ai/reset', auth.requireAuth, (ctx) => {
  const userId = ctx.req.user.id;
  const userAI = conversations.get(userId);
  if (userAI) userAI.clearHistory();
  return { success: true, message: 'Conversation reset' };
});
```

### Cost Management

```js
// Prompt length limit गर्ने
app.post('/api/ai/generate', auth.requireAuth, (ctx) => {
  const { prompt } = ctx.body;

  if (prompt.length > 1000) {
    return ctx.status(400).json({
      success: false,
      error: 'Prompt too long (max 1000 characters)'
    });
  }
  // ...
});
```

### Caching AI Responses

```js
const aiCache = new Map();

app.post('/api/ai/faq', async (ctx) => {
  const { question } = ctx.body;
  const cacheKey = question.toLowerCase().trim();

  // Cache hit
  if (aiCache.has(cacheKey)) {
    return { success: true, response: aiCache.get(cacheKey), cached: true };
  }

  const response = await ai.request(question, 'You are a FAQ bot.');
  aiCache.set(cacheKey, response);

  // 1 घन्टा पछि expire
  setTimeout(() => aiCache.delete(cacheKey), 60 * 60 * 1000);

  return { success: true, response, cached: false };
});
```

## Dolphin AI Architecture

```
dolphin chat command
        │
        ▼
DolphinAgent.interactiveChat()
        │
        ├── buildProjectMemory()     ← सबै files index गर्छ
        │       ├── walkDir()         ← files खोज्छ
        │       ├── updateSemanticIndex()  ← keywords index
        │       └── symbolIndex       ← functions/classes track
        │
        ├── loadHistory()            ← .dolphin/history.json पढ्छ
        │
        └── while(true) loop
                ├── readline input
                ├── semanticSearch()  ← relevant files find
                ├── AIService.requestStream()  ← AI call
                ├── parseAIResponse() ← tool calls extract
                └── executeTool()
                        ├── read   → fs.readFile
                        ├── write  → fs.writeFile (with permission)
                        ├── patch  → string replace (with permission)
                        ├── delete → fs.rm (with permission)
                        ├── shell  → execAsync (with permission + safety check)
                        └── done   → loop end
```
